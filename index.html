Docker Commands

Docker Management Commands

  • docker --version - Displays the installed Docker version.
  • docker info - Provides detailed information about the Docker installation.
  • docker help - Displays a list of Docker commands and their descriptions.

Container Management

  • docker run [options] IMAGE [COMMAND] [ARG...] - Runs a container based on the specified image.
  • docker ps [options] - Lists running containers.
  • docker start CONTAINER - Starts a stopped container.
  • docker stop CONTAINER - Stops a running container gracefully.
  • docker restart CONTAINER - Restarts a container.
  • docker kill CONTAINER - Forcefully stops a running container.
  • docker rm [options] CONTAINER - Removes a container.
  • docker logs [options] CONTAINER - Fetches logs from a container.
  • docker exec [options] CONTAINER COMMAND [ARG...] - Runs a command inside a running container.
  • docker attach CONTAINER - Attaches to a running container to view output or interact.
  • docker inspect CONTAINER - Displays detailed information about a container.
  • docker stats [CONTAINER] - Displays resource usage statistics for containers.
  • docker commit CONTAINER NEW_IMAGE - Creates a new image from a container's changes.

Image Management

  • docker images [options] - Lists all available images on the system.
  • docker pull IMAGE[:TAG] - Downloads an image from a Docker registry.
  • docker push IMAGE[:TAG] - Uploads a local image to a Docker registry.
  • docker rmi [options] IMAGE - Removes one or more images.
  • docker tag SOURCE_IMAGE[:TAG] TARGET_IMAGE[:TAG] - Tags an image with a new name or tag.
  • docker build [options] PATH | URL | - - Builds an image from a Dockerfile.
  • docker history IMAGE - Shows the history of an image.

Network Management

  • docker network ls - Lists all Docker networks.
  • docker network inspect NETWORK - Displays detailed information about a Docker network.
  • docker network create [options] NAME - Creates a new Docker network.
  • docker network connect NETWORK CONTAINER - Connects a container to a network.
  • docker network disconnect NETWORK CONTAINER - Disconnects a container from a network.

Volume Management

  • docker volume ls - Lists all Docker volumes.
  • docker volume create [options] NAME - Creates a new Docker volume.
  • docker volume inspect VOLUME - Displays detailed information about a volume.
  • docker volume rm VOLUME - Removes a volume.
  • docker volume prune - Removes all unused volumes.

System Commands

  • docker system df - Shows disk usage by Docker objects.
  • docker system prune [options] - Cleans up unused containers, networks, images, and volumes.
  • docker system info - Displays system-wide information about Docker.

Compose Commands (Docker Compose)

  • docker compose up [options] - Starts and runs multi-container applications.
  • docker compose down [options] - Stops and removes containers defined in the docker-compose.yml file.
  • docker compose ps - Lists containers managed by Docker Compose.
  • docker compose logs [options] - Displays logs from services defined in the docker-compose.yml.
